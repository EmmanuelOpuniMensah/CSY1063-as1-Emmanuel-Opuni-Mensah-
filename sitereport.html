Site Report

Technologies Used

I used a variety of technologies to build this site, including HTML and for the front-end. I also used Git for version control, and GitHub Pages to host the site. I used Visual Studio Code as my code editor.

Challenges Faced

While creating this portfolio website, one of the main challenges I faced was dealing with small mistakes , such as missing commas and incorrect syntax. These minor errors often caused my code to break, which was quite frustrating at times. I also struggled with implementing the CSS grid layout, as positioning elements correctly required a deeper understanding of how grid systems work. I struggles to keep up with time due to the fact that I have two jobs and I am a full time student, but I managed to complete the project on time I struglled creating more branches because they were not working as they should be.

Things I enjoyed

However, I enjoyed the research process and finding solutions to problems. I came across a piece of code online that helped me improve my grid structure, which I referenced appropriately. This allowed me to better understand how to create a more responsive and organised layout.One aspect I particularly enjoyed was working on the card background animation. Adding smooth transitions and glowing effects made my website feel more interactive and visually appealing. This experience not only improved my CSS skills but also allowed me to be more creative with my designs. Overall, this project has been a great learning experience, helping me strengthen my attention to detail and problem-solving skills. I will try and use more branches next since my branches did not function as they should have.
